Développement Web Fullstack : Jojo's Adventure

Vanilla JavaScript PHP Backend HTML5 / CSS3 Architecture MVC

L'Objectif Académique

Réalisé en 2ème année de Licence, ce projet avait pour but de maîtriser les fondamentaux du développement web sans l'utilisation de frameworks modernes (comme React ou Angular). L'objectif était de comprendre la mécanique interne du web : manipulation du DOM, gestion des événements et communication client-serveur.

J'ai choisi de développer un site encyclopédique interactif sur l'univers du manga Jojo's Bizarre Adventure, nécessitant une structure de données rigoureuse et une interface utilisateur dynamique.

Architecture & Fonctionnalités

Le site est structuré autour d'une navigation intuitive divisée en trois modules principaux, gérés dynamiquement par JavaScript et PHP.

1. Module Personnages

Système de fiches dynamiques. Au clic, le script charge les détails du protagoniste, ses statistiques, son histoire et ses anecdotes sans recharger la page complète (Simulation SPA).

2. Les Arcs Narratifs

Présentation structurée des différentes parties de l'anime. Utilisation de CSS Grid et Flexbox pour une mise en page responsive s'adaptant aux descriptifs de chaque saison.

3. Section "Secrets"

Une section dédiée aux anecdotes méconnues sur l'auteur Hirohiko Araki. Intégration PHP pour gérer l'affichage conditionnel de ces informations bonus.

Le Défi "Vanilla JS"

Pourquoi pas de Framework ?

Le défi technique principal était de tout coder en Javascript Natif (Vanilla). Cela m'a permis de comprendre comment fonctionnent réellement les écouteurs d'événements (Event Listeners), la manipulation du DOM et la gestion de l'asynchronisme, compétences souvent masquées par les bibliothèques modernes.

Résultat : Un site web léger, rapide et entièrement optimisé, avec un code source propre et commenté.
