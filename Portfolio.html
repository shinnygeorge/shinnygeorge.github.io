Projet Web Fullstack : Architecture & Interactivité

HTML • CSS • PHP • JS

Objectif Technique

Développement intégral d'une application web dynamique sans l'utilisation de frameworks (Vanilla JS/CSS). L'objectif était de maîtriser les interactions natives Front-end et la structuration Back-end (PHP).

Le thème (Jojo's Bizarre Adventure) a servi de support pour créer une architecture de contenu complexe.

Implémentation

  • Navigation SPA (Single Page Application) : Conception d'un système d'onglets en JavaScript pour basculer instantanément entre les vues (Personnages / Synopsis / Anecdotes) sans rechargement de page.
  • Manipulation du DOM : Création de fenêtres modales (Pop-ups) déclenchées par des événements onclick pour afficher les détails des 6 arcs narratifs.
  • Structuration des Données : Gestion de fiches techniques pour les personnages principaux et intégration d'une section "Secrets de conception" sur l'auteur.
